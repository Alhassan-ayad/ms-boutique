Paymob Sandbox

Testing steps and checkout entry point

Use this page to verify both order paths before launch. The sandbox checkout uses test Paymob credentials, while cash on delivery stays on the direct order flow.

Sandbox checkout flow

  1. Set your backend environment variables to Paymob sandbox values.
  2. Add one or more products to the cart.
  3. Open checkout with sandbox mode enabled.
  4. Select Pay with Paymob Sandbox and place the order.
  5. Complete the iframe with the test card details from your Paymob sandbox account.
  6. Confirm the payment callback marks the session paid and the order appears in the backend.

Cash on delivery check

  1. Open the regular checkout page.
  2. Select Cash on delivery.
  3. Place the order without opening Paymob.
  4. Verify the order is created immediately and the cart clears after success.
